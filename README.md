Authentication Module for Safan Framework
===============

REQUIREMENTS
------------
PHP > 5.3.0

safan-lab/gap >= 1.0

SETUP
------------

If you're using [Composer](http://getcomposer.org/) for your project's dependencies, add the following to your "composer.json":

```
"require": {
    "safan-lab/auth": "1.0.0-BETA"
}
```
